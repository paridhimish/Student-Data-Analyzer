 1. Student Data Analyzer

Description

A Python project that analyzes student marks and provides insights using data analysis and visualization.

Features

- Takes user input for student names and marks
- Calculates average, highest, and lowest marks
- Identifies top-performing student
- Visualizes data using bar graph
- Saves data to CSV file

Tech Stack

- Python
- Pandas
- Matplotlib

 2. Student Performance Predictor

Description

A Machine Learning model that predicts student final marks based on multiple factors.

Features

- Uses features like:
  - Study Hours
  - Attendance
  - Previous Marks
- Model: Random Forest Regressor
- Evaluated using accuracy score and MAE
- Takes dynamic user input for prediction

Tech Stack

- Python
- Pandas
- Scikit-learn

 3. Fake News Detector (NLP)

Description

An NLP-based project that classifies news as Real or Fake using text processing techniques.

Features

- Converts text into numerical form using TF-IDF
- Uses Logistic Regression for classification
- Takes user input (news text)
- Predicts whether news is Real or Fake

Tech Stack

- Python
- Scikit-learn
- NLP (TF-IDF Vectorizer)
